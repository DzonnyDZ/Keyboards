Keyboard layout pack "Ruská - české rozložení" (Russian - Czech layout) 1.2.0
Contains three keyboard layouts:
CZRuQZNm - "Ruská - české rozložení QWERTZ - numerická řada" (Russian - Czech layout QWERTZ (ЮЪЭРТЗ) with numeric row)
CZRuQZCp - "Ruská - české rozložení QWERTZ - velká písmena místo numerické řady" (Russian - Czech layout QWERTY (ЮЪЭРТЗ) with capital letters instead numeric row)
CZRuQYNm - "Ruská - české rozložení QWERTY - numerická řada" (Russian - Czech layout QWERTZ (ЮЪЭРТЫ) with numeric row)

To install:
Use the setup.exe file from appropriate folder.
Tip: See layout images in individual foldres to decide which layouts suits you the best.
- Previous version of product "Ruská - rozložení jako latinka" (Russian - latin-like layout) does NOT have to be uninstalled, however it's recommended to uninstall it.
- These three layouts can be installed simultaneously.

To ininstall:
Use the Add/Remove programs control panel.

Supported operationg systems:
Win NT 4.0, Win 2000, Win XP, Win 2003, Win Vista, Win 2008, Win 7

Supported platforms:
x86 (32 bit), x64 (64 bit), Itanium (64 bit)
